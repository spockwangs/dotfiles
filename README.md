dotfiles
========

## Install

Clone to local machine.

    $ git clone git://github.com/spockwangs/dotfiles.git

Copy to home directory. For Linux it is

    $ cd dotfiles
    $ ./install.sh

and for Windows

    $ cd dotfiles
    $ ./install.bat

## Make local customizations

Put your customizations in dotfiles appended with `.local`:

* `~/.bashrc.local`
* `~/.gitconfig.local`
* `~/.tmux.conf.local`
